# proyecto-AE
Este repositorio contiene toda la información referente al proyecto para Aprendizaje Estadistico de Daniel José Duque y Tomas Gómez. El objetivo prinicpal será replicar los espectros infrarrojos del cristal de B-hematina empleando el modelo de Batiata et al. con un énfasis en el entendimiento del fine-tuning.
