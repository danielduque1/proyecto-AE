# proyecto-AE
Este repositorio contiene toda la información referente al proyecto de Aprendizaje Estadistico, Instituto de Física, UdeA, por Daniel José Duque y Tomas Gómez. El objetivo prinicpal será replicar los espectros infrarrojos del cristal de B-hematina empleando el modelo de Batiata et al. con un énfasis en el entendimiento del fine-tuning.
